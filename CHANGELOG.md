# Change Log

All notable changes to the "dev-pocket" extension will be documented in this file.

## [0.0.2] - 2025-05-16

### Added
- URL encoding and decoding
- SQL `WHERE IN` clause quotes in multiline string
- Right-click context menu integration for all commands

## [0.0.1] - 2025-05-02

### Added
- ENV/JSON format conversion with auto-detection
- Base64 encoding and decoding
- JSON minification for single and multi-object JSON
- Unix timestamp conversion with GMT, local time, and relative time display
- Unix timestamp hover provider for inline conversions
